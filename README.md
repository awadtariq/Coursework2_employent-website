# Job Finder - Professional Job Tracking System

A modern, web-based job management platform developed as part of the CS5003 coursework. This application allows users to manage job listings, track opportunities, and communicate via a built-in messaging system.

## 🚀 Features
- **Secure Authentication:** User registration and login with session management.
- **Job Database:** Full CRUD (Create, Read, Update, Delete) functionality for job postings.
- **AJAX Search:** Real-time job filtering and suggestions using JSF AJAX.
- **Job Tracking:** Users can "Track" specific jobs to save them to a personal dashboard.
- **Contact System:** Integrated messaging for user inquiries.
- **Responsive Design:** Modern UI with a professional sidebar and glassmorphism effects.

## 🛠️ Technology Stack
- **Frontend:** Jakarta Server Faces (JSF) 4.0, Facelets, CSS3 (Modern Grid/Flexbox).
- **Backend:** Java (Jakarta EE), CDI (Contexts and Dependency Injection).
- **Persistence:** Jakarta Persistence API (JPA), Hibernate.
- **Database:** Apache Derby (JavaDB).
- **Server:** GlassFish 7.x.

## 📸 UI Improvements
