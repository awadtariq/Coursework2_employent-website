Who We Are

Job Finder is a web-based job tracking system developed as part of the CS5003 coursework project. The platform allows users to browse available job opportunities, track jobs, and manage tracked jobs.

What You Can Do

  • Register and log in securely
  • Browse available jobs
  • Search using AJAX-powered filtering
  • Track jobs you are interested in
  • Create, edit, and delete job listings

Technology Used

  • Jakarta Server Faces (JSF)
  • Jakarta Persistence API (JPA)
  • GlassFish Server
  • Apache Derby (JavaDB)

This project was developed as a group coursework submission for CS5003.
